# Portafolio Pixelworks

Landing page creada con React y Vite para una simulacion de aplicacion laboral a Pixelworks, una agencia digital guatemalteca enfocada en productos web, SPAs, APIs REST y entornos Docker.

## Stack

- React para una SPA modular con componentes reutilizables.
- Vite como bundler por su arranque rapido y build sencillo.
- CSS organizado por secciones, con variables, responsive design y estados accesibles.
- ESLint configurado desde el inicio para mantener consistencia.

## Secciones incluidas

- Sobre mi, formacion y contacto.
- Tecnologias utilizadas, con explicacion de la selección.
- Proyectos con enlaces funcionales a repositorios publicos.
- Evidencia tecnica alineada a lo solicitado por Pixelworks.

## Instalacion

```bash
npm install
npm run dev
```

## Levantar con Docker

Para construir la app y levantarla en una sola linea:

```bash
docker compose up --build
```

Luego abre exactamente esta URL:

```bash
http://localhost:8080
```

Para detener el contenedor:

```bash
docker compose down
```
